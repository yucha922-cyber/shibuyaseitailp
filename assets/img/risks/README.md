# 放置リスクセクション 背景画像

このディレクトリには、「その不調、放っておくと『習慣』になります。」セクションの**背景画像**を配置します。

## 画像の配置方法

1. このディレクトリに **`bg.jpg`** というファイル名で画像をアップロードしてください
2. アップロードと同時に、自動でセクションの背景に表示されます
3. HTMLやCSSを編集する必要はありません

### GitHub Web画面からアップロードする手順

1. GitHub のリポジトリページで `assets/img/risks/` ディレクトリを開く
2. 右上の「Add file」→「Upload files」をクリック
3. お手持ちの画像ファイルをドラッグ＆ドロップ
4. **ファイル名が `bg.jpg` であることを確認**（違う場合はリネーム）
5. 「Commit changes」をクリック

## 推奨画像仕様

| 項目 | 推奨 |
|---|---|
| ファイル名 | `bg.jpg` |
| 形式 | JPG / WebP |
| アスペクト比 | **16:9 推奨**（横長画像） |
| 推奨幅 | 1600〜1920px |
| ファイルサイズ | **400KB以下推奨** |
| 構図イメージ | 慢性的疲労感／PC作業／時計／長時間労働 等の暗めの雰囲気が合います |

## 仕様

- 画像の上に**深いネイビーの半透明オーバーレイ（82〜88%）**が自動で重なり、白文字が読みやすくなります
- 3つのリスクカードは**フロストガラス風**で背景画像の上でも常に視認可能
- 画像が未配置の場合は `placeholder.svg`（ダークネイビー〜ティールのグラデーション）が自動表示

## 別の画像形式・ファイル名を使いたい場合

`index.html` 内の以下を編集してください：

```html
<img
  src="assets/img/risks/bg.jpg"  ← ここを変更
  ...
  onerror="this.onerror=null;this.src='assets/img/risks/placeholder.svg';">
```
