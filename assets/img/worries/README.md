# 共感セクション 背景画像

このディレクトリには、「こんなお悩み、ありませんか？」セクションの**背景画像**を配置します。

## 画像の配置方法

1. このディレクトリに **`bg.jpg`** というファイル名で画像をアップロードしてください
2. アップロードと同時に、自動でセクションの背景に表示されます
3. HTMLやCSSを編集する必要はありません

### GitHub Web画面からアップロードする手順

1. GitHub のリポジトリページで `assets/img/worries/` ディレクトリを開く
2. 右上の「Add file」→「Upload files」をクリック
3. お手持ちの画像ファイルをドラッグ＆ドロップ
4. **ファイル名が `bg.jpg` であることを確認**（違う場合はリネーム）
5. 「Commit changes」をクリック

## 推奨画像仕様

| 項目 | 推奨 |
|---|---|
| ファイル名 | `bg.jpg` |
| 形式 | JPG / WebP |
| アスペクト比 | **16:9 推奨**（横長画像） |
| 推奨幅 | 1600〜1920px |
| ファイルサイズ | **400KB以下推奨** |
| 構図 | デスクワーク・肩こり・疲労感などのイメージ／顔は写さない or ぼかす |

## 仕様

- 画像の上に **ネイビーの半透明オーバーレイ**（78〜85%）が自動で重なり、白文字が読みやすくなります
- お悩みカードは**フロストガラス風**（白半透明＋ブラー）で背景画像との相性に関わらず常に視認可能
- 画像が未配置の場合は `placeholder.svg`（ネイビー〜ティールのグラデーションプレースホルダー）が自動表示

## 別の画像形式・ファイル名を使いたい場合

`index.html` 内の以下を編集してください：

```html
<img
  src="assets/img/worries/bg.jpg"  ← ここを変更
  ...
  onerror="this.onerror=null;this.src='assets/img/worries/placeholder.svg';">
```
